# WLAN Network Modeling Workshop 

## Overview:

The WLAN Network Modeling Workshop provides a hands-on introduction to end-to-end WLAN modeling using MATLAB. Participants learn to model and simulate Wi-Fi 7 (802.11be) and earlier networks, study WLAN–Bluetooth coexistence, and visualize network performance in realistic scenarios with minimal coding.
## What You Will Learn

- Understand WLAN network simulation using MATLAB  
- Simulate multi-node WLAN systems across APP, MAC, and PHY layers  
- Visualize node states and capture key network statistics  
- Analyze the impact of Bluetooth and other networks on performance  
- Explore Wi-Fi 7 multi-link operations (eMLSR and STR)
## Key Highlights
- Guided, step-by-step exercises
- Minimal coding effort required
- Focus on practical, real-world scenarios
## Target Audience

- WLAN system engineers  
- WLAN PHY/MAC engineers  
- Professionals interested in WLAN and hybrid (WLAN–Bluetooth) network simulation using MATLAB  
## Repository Overview

This repository contains the files required to run a hands-on WLAN network modeling workshop in MATLAB®.

- **Exercise files**: Require code additions to run  
- **Solution scripts**: Include completed code and run end-to-end  

### Exercises Included

- **Exercise 1** – Run the simulation engine, configure WLAN nodes, build a simple network, and view statistics  
- **Exercise 2** – Configure traffic models, QoS, and model interference  
- **Exercise 3** – Implement a custom channel model for WLAN networks  
- **Exercise 4** – Simulate a network with multi-link operation (MLO) nodes (Wi-Fi 7)  

## Setup

To Run:

- Use the **Exercise** files to perform the hands-on workshop  
- To run completed examples, open and execute the following files in MATLAB:

  - `EXPL1_WLAN_Simplest_Network.mlx`  
  - `EXPL2_WLAN_QoS_Interference.mlx`  
  - `EXPL3_WLAN_CustomChannelModel.mlx`  
  - `EXPL4_WLAN_MLO.mlx`
